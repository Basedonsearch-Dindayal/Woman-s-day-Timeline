# 🌸 Women's Day Timeline

A beautifully crafted, interactive timeline celebrating the achievements, history, and milestones of women throughout the years. Built with [Next.js](https://nextjs.org), React, and Tailwind CSS.

## ✨ Features

- **Interactive Timeline**: Scroll to explore key historical events.
- **Modern UI**: Styled gracefully with Tailwind CSS.
- **Responsive Design**: Works perfectly on mobile and desktop devices.
- **Fast Performance**: Powered by Next.js app router.

## 🚀 Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the timeline.
